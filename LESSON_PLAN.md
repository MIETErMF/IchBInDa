# План уроків: Проєкт «IchBInDa»

## Опис проєкту

«IchBInDa» (нім. «Я тут») — застосунок для мотивації підлітків. Щодня генерує 4 персональні завдання за допомогою штучного інтелекту:
- 2 прості на сьогодні
- 1 виклик на тиждень
- 1 ціль на місяць

## Що вже зроблено

- Генерація завдань через OpenAI API
- Показ завдань на красивому фоні (Unsplash)
- База даних з таблицею achievements
- Базовий інтерфейс на Bootstrap

---

## Інтерактивність

| Урок | Тема | Що робимо |
|------|------|-----------|
| 1 | Кнопка «Виконано» | Додати чекбокс/кнопку біля кожного завдання для позначення виконання |
| 2 | AJAX-запити | Зробити щоб сторінка не перезавантажувалась при натисканні (fetch/axios) |
| 3 | Візуальний фідбек | Перекреслювати виконані завдання, змінювати колір, додати анімацію |
| 4 | Дата виконання | Зберігати коли саме завдання було виконано (нове поле в БД) |
| 5 | Статистика | Показати «Виконано 3 з 4 сьогодні» |

**Учень вивчає:** JavaScript, AJAX, робота з DOM
